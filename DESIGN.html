Database Schema

string — bắt buộc, không được null
string? — nullable, có thể để trống
FK → — khoá ngoại liên kết bảng khác
int PK — khoá chính, tự tăng

Users

AspNetUsers (extends IdentityUser → ApplicationUser)
  + FullName       string
  + DateOfBirth    DateTime?
  + AvatarUrl      string?
  + StudentId      string?   -- chỉ dành cho Tenant

Tài sản

Buildings
  Id            int PK
  LandlordId    FK → AspNetUsers
  Name          string
  Address       string
  Province      string
  District      string
  Ward          string
  Description   string?
  CreatedAt     DateTime

Rooms
  Id            int PK
  BuildingId    FK → Buildings
  RoomNumber    string
  FloorNumber   int?
  Area          decimal   -- m²
  Capacity      int?      -- null = không giới hạn số người thuê
  Description   string?
  Status        enum: Available | Occupied | Maintenance | Inactive
  CreatedAt     DateTime

RoomImages
  Id                  int PK
  RoomId              FK → Rooms
  ImageUrl            string   -- Cloudinary URL
  CloudinaryPublicId  string
  IsPrimary           bool
  SortOrder           int

Amenities  -- danh mục tiện ích (WiFi, Điều hoà, Máy giặt, Bãi xe…)
  Id         int PK
  Name       string
  IconClass  string   -- CSS icon class

RoomAmenities  -- many-to-many, composite PK (RoomId, AmenityId)
  RoomId     FK → Rooms
  AmenityId  FK → Amenities
📌 RULE — Room.Status: Tự động cập nhật theo Contract active — không toggle thủ công. Có Contract.Status = Active → Occupied; không có → Available.

Cấu hình phí Điểm khác biệt

FeeConfigs
  Id            int PK
  BuildingId    FK → Buildings  (nullable)
  RoomId        FK → Rooms      (nullable) ← override cấp phòng
  Name          string          -- VD: "Tiền điện", "Phí xe máy"
  FeeCategory   enum: Rent | Electricity | Water | Internet | Parking | Other
  CalcType      enum: Fixed | PerUnit
  UnitPrice     decimal
  Unit          string          -- "tháng", "kWh", "m³", "xe"…
  IsActive      bool
  SortOrder     int
📌 RULE — Ưu tiên khi tạo hoá đơn: RoomId-level override BuildingId-level cùng FeeCategory. Lấy tất cả FeeConfig IsActive = true áp dụng cho phòng đó.

Lịch hẹn xem phòng

ViewingRequests
  Id              int PK
  RoomId          FK → Rooms
  TenantId        FK → AspNetUsers
  ProposedTime    DateTime    -- tenant đề xuất
  ConfirmedTime   DateTime?   -- chủ đổi giờ (nullable)
  Status          enum: Pending | Confirmed | Rescheduled | Cancelled | Completed
  TenantNote      string?
  LandlordNote    string?
  CreatedAt       DateTime
📧 RULE — Email: Gửi email cho Tenant khi Status chuyển sang Confirmed hoặc Rescheduled.

Hợp đồng

Contracts
  Id                 int PK
  RoomId             FK → Rooms
  TenantId           FK → AspNetUsers   -- người ký HĐ (primary)
  StartDate          DateTime
  EndDate            DateTime?           -- null = open-ended
  ActualEndDate      DateTime?           -- ngày thực tế kết thúc
  Status             enum: Pending | Active | Expired | Terminated
  DepositAmount      decimal
  MonthlyRent        decimal             -- tổng tiền phòng, không tự đổi khi ghép
  SeekingRoommates   bool               -- đang tìm người ghép
  DesiredOccupancy   int?               -- muốn ở bao nhiêu người tổng (null = không giới hạn)
  GeneratedDocUrl    string?
  TerminationReason  string?
  Notes              string?
  CreatedAt          DateTime

ContractMembers   -- người ở ghép, đăng ký vào hệ thống (không ký HĐ riêng)
  Id           int PK
  ContractId   FK → Contracts
  TenantId     FK → AspNetUsers
  JoinDate     DateTime
  LeaveDate    DateTime?   -- null = đang ở
  Note         string?
👥 RULE — Occupancy: Tổng người ở = 1 (primary) + ContractMembers có LeaveDate = null. Phải ≤ Room.Capacity (nếu có) và ≤ Contract.DesiredOccupancy (nếu có).
💰 RULE — Tiền phòng: MonthlyRent là giá tổng phòng, không tự thay đổi khi người ghép vào/ra. Việc chia tiền nội bộ do các tenant thoả thuận với landlord.
🔍 RULE — Tìm ghép: Hiển thị "đang tìm ghép" khi SeekingRoommates = true. Số chỗ còn: DesiredOccupancy - current occupants.
📌 RULE — Lịch sử: Không xoá contract cũ, chỉ đổi Status. Khi member rời đi, set LeaveDate — không xoá dòng.
📄 RULE — Template: Chỉ dùng template mặc định của app (không cho upload custom). Xuất cả PDF lẫn Word.

Hoá đơn & Thanh toán

Invoices
  Id            int PK
  ContractId    FK → Contracts
  BillingYear   int
  BillingMonth  int      -- 1–12
  TotalAmount   decimal
  Status        enum: Draft | Sent | Paid | Overdue | Cancelled
  DueDate       DateTime
  Notes         string?
  CreatedAt     DateTime

InvoiceItems
  Id                int PK
  InvoiceId         FK → Invoices
  FeeConfigId       FK → FeeConfigs  (nullable — có thể là dòng thủ công)
  Description       string
  Quantity          decimal?  -- null nếu Fixed
  UnitPrice         decimal
  Amount            decimal
  PreviousReading   decimal?  -- cho PerUnit (điện/nước)
  CurrentReading    decimal?

Payments
  Id              int PK
  InvoiceId       FK → Invoices
  Amount          decimal
  PaymentDate     DateTime
  Method          enum: Cash | BankTransfer | MoMo | VNPay
  TransactionRef  string?
  Notes           string?
  RecordedById    FK → AspNetUsers
  CreatedAt       DateTime
🧮 RULE — Tính tiền: Amount = Quantity × UnitPrice (PerUnit) hoặc UnitPrice (Fixed).
➗ RULE — Phòng ghép: Hoá đơn điện/nước chia đều tự động 1/N người đang active trong phòng.

Đánh giá & Thông báo

RoomReviews
  Id           int PK
  RoomId       FK → Rooms
  ReviewerId   FK → AspNetUsers
  ContractId   FK → Contracts   -- xác minh đã từng ở phòng đó
  Rating       int              -- 1–5
  Content      string?
  IsApproved   bool             -- admin/landlord có thể ẩn
  CreatedAt    DateTime

Notifications
  Id                  int PK
  UserId              FK → AspNetUsers
  Type                enum: ViewingConfirmed | ViewingRescheduled | ViewingCancelled
                             | InvoiceDue | ContractExpiring | ContractSigned
                             | NewViewingRequest | PaymentReceived | NewReview
  Title               string
  Content             string
  IsRead              bool
  RelatedEntityType   string?   -- "Contract", "Invoice", "ViewingRequest"…
  RelatedEntityId     int?
  CreatedAt           DateTime
